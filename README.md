# Filament_1

本仓库用于 kHz 激光空气成丝数值仿真。

## 主要入口

- 运行与参数中文指南：`Filament_python/README.md`
- 默认示例配置：`Filament_python/config.json`
- 另一份较大网格配置：`Filament_python/khz_config.json`

## 快速开始

```bash
pip install -r Filament_python/requirements.txt
python -m Filament_python.KHz_filament.cli Filament_python/config.json
```
